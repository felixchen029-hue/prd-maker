# 云计算 IaaS 领域检查表

## 计算实例

- 实例族、规格、CPU 架构、vCPU、内存、本地盘和加速器。
- 创建、启动、停止、重启、关机、释放、回收和故障迁移状态。
- 镜像来源、版本、共享、复制、导入导出和安全扫描。
- 密钥、密码、初始化脚本、元数据和登录方式。
- 批量操作、幂等性、异步任务、超时和错误码。

## 关联资源

- VPC、子网、安全组、弹性网卡、公网 IP、带宽和 DNS。
- 系统盘、数据盘、快照、备份、加密和挂载限制。
- 区域、可用区、资源库存、配额、容量和调度约束。
- IAM、服务角色、最小权限、审计日志和操作留痕。

## 商业与运营

- 按量、包周期、竞价、预留或承诺用量。
- 计费开始与停止时点、变配、续费、欠费、退款和账单拆分。
- 配额申请、容量保障、售罄、白名单和灰度。
- SLA、可用性、性能、故障恢复和客户支持边界。

## 产品界面

- 控制台、OpenAPI、CLI、SDK 和 IaC 是否一致。
- 状态刷新、进度反馈、批量操作和失败重试。
- 错误信息是否给出原因、影响和下一步动作。
- 指标、日志、事件、告警和成本可观测性。

## 竞品比较维度

- 产品定位和目标客户。
- 实例家族覆盖、规格灵活性和上新节奏。
- 购买路径、计费模式和价格透明度。
- 网络、存储、镜像、运维与安全集成。
- API 完整度、控制台体验和生态兼容。
- 区域覆盖、容量保障、SLA 和支持体系。

## PRD 最低验收要求

- 每个功能有角色、前置条件、主路径、异常路径和结果。
- 生命周期状态转移明确，禁止的转移有错误反馈。
- 权限、审计、计费和配额行为可验证。
- API 与控制台字段、枚举和错误语义一致。
- 灰度、回滚、数据迁移和存量实例影响有说明。
